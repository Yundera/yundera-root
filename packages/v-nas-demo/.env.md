# Environment Configuration

Copy and paste the following block into your `.env` file:

```env
# Scaleway Configuration
SCW_API_URL=https://api.scaleway.com
SCW_ACCESS_KEY=SCWxxx
SCW_SECRET_KEY=43xxx
SCW_DEFAULT_ORGANIZATION_ID=6cxxx
SCW_DEFAULT_PROJECT_ID=xxxx
SCW_ZONE=pl-waw-1
SCW_INSTANCE=DEV1-S
SCW_IMAGE=xxxx

# SSH Configuration
SSH_KEY="-----BEGIN OPENSSH PRIVATE KEY-----
xxxx
-----END OPENSSH PRIVATE KEY-----"

# API Keys
SERVICE_API_KEY=xxxxxx
SENDGRID_API_KEY=xxxx

# Email Configuration
SENDMAIL_FROM_EMAIL=xxxx

# Mesh Router Configuration
MESH_ROUTER_BACKEND_URL=https://nsl.sh/router/api
MESH_ROUTER_BACKEND_API_KEY=xxxxxxxx
```

Remember to replace all `xxxx` placeholders with your actual values and never commit the `.env` file to version control.
